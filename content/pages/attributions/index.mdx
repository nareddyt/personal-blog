Creating this website was a challenge as I do NOT work with frontend normally.
Luckily, the open source community put together plenty of helpful resources.
I created this website in one week thanks to the following:

## Technology

- Core Framework: [Gatsby](https://www.gatsbyjs.com/)
- Interactive UI: [React.js](https://reactjs.org/)
- Styling: [Theme UI](https://theme-ui.com/)
- Hosting: [Gatsby Cloud](https://www.gatsbyjs.com/products/cloud/).
- Icon Resizing: [Real Favicon Generator](https://realfavicongenerator.net/)

## Creative Works
